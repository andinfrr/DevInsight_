Pengenalan Tabel pada HTML

Membuat Tabel Pada HTML

Tabel merupakan struktur yang digunakan untuk menampilkan informasi dalam bentuk baris dan kolom. Dalam HTML, pembuatan tabel dilakukan menggunakan beberapa tag, yaitu:

  • <table> : Tag utama untuk membuat tabel.
  • <tr> : Table Row, digunakan untuk membuat baris dalam tabel.
  • <td> : Table Data, digunakan untuk menampilkan data dalam kolom.
  • <th> : Table Header, digunakan untuk bagian kepala (judul kolom) tabel.

    • Cara Penggunaan Tabel Pada HTML:

    <table border="1">
    <tr>
    <th>No</th>
    <th>Nama</th>
    <th>Alamat</th>
    </tr>
    <tr>
    <td>1</td>
    <td>Badrul</td>
    <td>Sumedang</td>
    </tr>
    <tr>
    <td>2</td>
    <td>Andi</td>
    <td>Jakarta</td>
    </tr> </table>
