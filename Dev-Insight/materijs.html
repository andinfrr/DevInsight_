Pengenalan JavaScript Dasar

Apa itu JavaScript?

Javascript merupakan bahasa pemrograman yang berguna untuk membuat interaksi pada website. sehingga website terlihat lebih hidup. seperti misalnya membuat efek-efek pada website. javascript sudah menjadi salah satu bahasa pemrograman yang wajib dan harus di kuasai untuk membangun website yang modern.

Cara Penggunaan Kode JavaScript bisa ditulis langsung di file HTML menggunakan tag <script>, atau dipisah di file .js eksternal.
Contoh:

<script> alert("Halo, Dunia!"); </script>

Penggunaan Variabel dan Operasi Aritmatika

1. Variabel

Variabel digunakan untuk menyimpan nilai. Dideklarasikan dengan var, let, atau const.
Contoh:

<script>
var nama = "Malas Ngoding";
var alamat = "Jakarta";
document.getElementById("nama").innerHTML = nama;
document.getElementById("alamat").innerHTML = alamat;
</script>

2. Operasi Aritmatika

Operator aritmatika digunakan untuk melakukan operasi matematis pada nilai (angka). Berikut adalah daftar operator aritmatika yang umum digunakan di JavaScript:

Tag Fungsi
+ Digunakan untuk Penjumlahan
- Digunakan untuk Pengurangan
* Digunakan untuk Perkalian
/ Digunakan untuk Pembagian
% Modulus
++ Digunakan untuk menambah nilai variabel sebanyak 1 (Increment).
-- Digunakan untuk mengurangi nilai variabel sebanyak 1. (Decrement)
