Pengenalan CSS Dasar Mulai dan Jenis-Jenis CSS

Apa itu CSS?

CSS merupakan singkatan dari ” Cascading Style Sheets “. sesuai dengan namanya CSS memiliki sifat ” style sheet language ” yang berarti bahasa pemrograman yang di gunakan untuk web design. CSS adalah bahasa pemrograman yang di gunakan untuk men-design sebuah halaman website.
Ada tiga cara utama untuk menambahkan CSS ke HTML Anda. Mari kita lihat masing-masing:

  • CSS Inline
  • CSS Internal
  • CSS Eksternal

    • Jenis-Jenis CSS

    1. CSS Inline

    Cara menggunakan CSS Inline itu dengan cara menambahkan langsung di tag HTML.
    Contohnya: <h1 style="color:blue;">Judul Biru </h1>

    2. CSS Internal

    Cara menggunakan CSS Internal itu dengan cara menambahkan langsung di tag <head>.
    Contohnya: <style body> { background: powderblue; } h1 { color: blue; } </style>

    3. CSS Eksternal

    File css di simpan dengan ekstensi .css. kemudian di import atau di hubungkan kedalam file HTML atau PHP yang ingin kita design dengan CSS menggunakan syntax berikut ini.
    Contohnya: <link rel="stylesheet" href="styles.css">

    Mengenal Jenis-Jenis Seclector

    Ada banyak sekali jenis selector. Berikut ini beberapa jenis selector yang umum digunakan dan juga yang akan dipakai dalam CSS Journey ini:

    1. Universal

    Memilih semua elemen dengan selector: *

    2. Elemen

    Memilih semua elemen tertentu, contoh: p memilih semua elemen paragraf

    3. Kelas

    Memilih elemen dengan class tertentu, contoh: .classname

    4. ID

    Memilih elemen dengan ID tertentu, contoh: #idname

    5. Atribut

    Memilih elemen yang memiliki atribut tertentu, contoh:[attribute]

    6. Grup

    Memilih elemen yang cocok dengan beberapa selector, contoh: selector1, selector2

    7. Pseudo-class

    Memilih elemen dalam keadaan tertentu, contoh: :hover

    8. Pseudo-element

    Memilih bagian tertentu dari elemen, contoh: ::first-line

    9. Descendant

    Memilih semua elemen p yang berada di dalam div, contoh: div p

    10. Child

    Memilih elemen p yang menjadi anak langsung dari div, contoh: div > p

    11. Adjacent Sibling

    Memilih elemen p yang langsung setelah div, contoh: div + p

    12. General Sibling

    Memilih semua elemen p yang bersaudara dengan div, contoh: div ~ p

    Mengenal Properties dalam CSS

    CSS memiliki banyak properti yang bisa Anda mainkan. Berikut adalah beberapa yang umum:

    1. Color

    Untuk mengatur warna dalam CSS menggunakan color: #ff0000;

    2. Font Size

    Untuk mengatur size dalam CSS menggunakan font-size: 16px;

    3. Margin

    Untuk mengatur ruang dalam elemen menggunakan margin: 10px;

    4. Padding

    Untuk mengatur ruang di dalam elemen padding: 5px 10px;

    2. Border

    Untuk Menambahkan border border: 1px solid black;
