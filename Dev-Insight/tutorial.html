Materi HTML

Pengenalan HTML

Memperkenalkan konsep dasar HTML: struktur dokumen, tag penting (seperti html, head, body), serta cara menampilkan teks, gambar, maupun link.

Latihan HTML dasar

Tutorial dasar HTML: membahas struktur halaman, elemen dan tag utama, penggunaan atribut seperti class dan id, serta cara menulis kode di editor.

Materi dan Tutorial HTML

Tutorial HTML untuk pemula mulai dari membuat file hingga memahami elemen dan atribut.

Materi CSS

Pengenalan CSS

Membahas konsep lanjutan CSS seperti Flexbox dan Grid, lengkap dengan demo praktis membuat layout responsif dari awal hingga selesai.

Latihan CSS Dasar

menjelaskan pengertian CSS, penggunaannya bersama HTML, serta contoh penerapannya untuk membangun halaman web yang menarik.

Materi dan Tutorial CSS

pengenalan sintaks, cara menyisipkan ke HTML, serta contoh styling seperti warna, font, margin, hingga layout sederhana.

Materi JavaScript

Pengenalan JavaScript

pengenalan sintaks, variabel, tipe data dasar, integrasi ke HTML, dan cara menjalankan kode di browser.

Latihan JS Dasar

belajar JavaScript dari awal selama 1 jam: mulai dari sintaks dasar, variabel, tipe data, hingga manipulasi elemen web menggunakan DOM.

Materi dan Tutorial JS

tutorial lengkap JavaScript dasar selama 8 jam mulai dari sintaks dasar, variabel, tipe data, operator, fungsi, struktur kontrol, hingga manipulasi DOM.
